 🎓 Student Scholarship Portal

A full-stack web application developed to simplify and digitize the scholarship application process for students, institutes, and administrators.


 Project Description

The **Student Scholarship Portal** is a centralized system that allows students to apply for scholarships online, track their application status, and enables institutions and administrators to manage and verify applications efficiently.

This system reduces paperwork, improves transparency, and ensures faster processing of scholarship applications.


 👨‍🎓 Student Module

* Student Registration & Login
* View Available Scholarships
* Apply for Scholarships
* Track Application Status

🏫 Institute Module
* Institute Login
* View Student Applications
* Approve / Reject Applications

🏛️ Admin Module
* Add / Manage Scholarship Schemes
* Manage Students & Institutes
* Final Approval & Fund Allocation



 Frontend:
* HTML5
* CSS3
* JavaScript

Backend: Java (Servlets & JSP)

 Database: MySQL

 Server: Apache Tomcat

 Tools:Eclipse IDE
        Git & GitHub


 📊 System Architecture

The system follows a 3-tier architecture:

1. Presentation Layer (Frontend)
2. Business Logic Layer (Java Servlets/JSP)
3. Data Layer (MySQL Database)

 ⚙️ How It Works

1. Student registers and logs into the portal
2. Views available scholarships
3. Submits application form with required details
4. Data is stored in MySQL database
5. Institute verifies the application
6. Admin gives final approval
7. Student can track application status
